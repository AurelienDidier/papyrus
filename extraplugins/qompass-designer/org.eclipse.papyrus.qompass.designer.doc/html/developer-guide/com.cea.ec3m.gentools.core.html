org.eclipse.papyrus.qompass.designer.core

This is the main Qompass plugin that contains the dialogs and model transformations code as well as a set of utilities. Qompass consists of a set of package (ccegc is an abbreviation for the patch org.eclipse.papyrus.qompass.designer.core):
  • ccegc.acceleo a driver for the versoin 2.7.x of Acceleo. This driver is used to instantiate text templates that refer to model elements. These are primarily used for the connector instantiation (since the implementations given in form of Opaque behaviors need to be instantiated as well.
  • ccegc.deployment
  • ccegc.transformation Main transformation code
  • ccegc.synchronization A part of the utilities that synchronize derived elements, in particular the implementation of a component which has for instance
    • implement the interfaces provided by a component port
